## 🏗️ 為什麼做這個？ (The Concept)

幾年前，我曾做過一個版本的部落格（存放在[這裡](https://github.com/jason111nn/jason111nn-old-blog)）。

回頭看那個版本，覺得當時的技術與設計還點幼稚，因此決定重新啟動這個專案。我希望能利用這幾年所學的開發經驗，打造一個符合現階段美感與技術標準的全新版本。

好好利用我的所學，呈現出我心目中的網站 q(≧▽≦q)

---

## 🛠️ 技術點 (Tech Stack)

本專案採用 2026 年主流的開發工具鏈，追求簡潔與極速的效能：

- **核心 (Core)**: **Vue 3.5+** - 所有的邏輯起點。
- **建置 (Build)**: **Vite 8** - 目前最快的建置工具，提供極佳的開發體驗。
- **樣式 (Styling)**: **Tailwind CSS v4** - 現代化的 CSS-first 架構，保持代碼乾淨、執行極速。
- **動態 (Animation)**: **GSAP 3.15+** - 負責處理精細的物理動效與時間軸控制。
